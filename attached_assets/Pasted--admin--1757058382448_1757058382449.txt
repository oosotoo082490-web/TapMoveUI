목표
	•	세미나 신청과 제품 결제를 로그인 없이 진행 가능하도록 변경
	•	상단 네비게이션 “로그인”을 **“관리자”**로 표기 변경
	•	로그인은 관리자 영역(/admin) 에서만 필요
	•	제품 상세 페이지에서 “로그인 필요”/“로그인하러 가기” 제거 → “결제하기” 즉시 실행

⸻

1) 네비게이션 변경
	•	상단 네비게이션에서 **“로그인” 텍스트를 “관리자”**로 바꿔라.
	•	링크는 기존 로그인 경로 대신 /admin/login 으로 연결한다.
	•	모바일 네비/드로어에도 동일 반영.

2) 라우트 접근 권한 정책 수정
	•	인증 미들웨어/가드(예: middleware.ts, withAuth, ProtectedRoute)에서
	•	보호 대상: /admin/** 만 보호
	•	비보호 대상: /seminar/**, /products/**, /checkout/**, /orders/** 등 구매/신청 라우트는 전부 공개
	•	코드 내 requireLogin, signIn(), “로그인 필요” 안내가 구매/신청 경로에서 한 줄도 남지 않도록 전부 제거한다.
	•	기존 인증 로직(NextAuth/Firebase/Supabase 등)은 유지하되, 관리자 진입시에만 강제한다.

3) 세미나 신청 폼 공개 처리
	•	세미나 신청 페이지(/seminar/apply 등)를 공개 페이지로 전환.
	•	로그인 유도 문구/버튼을 전부 삭제.
	•	제출 시 DB 저장 + 알림(기존에 연결된 SMS/메일 있으면 그대로 유지).
	•	Form 필수값: 성명, 생년월일, 이메일, 연락처, 주소, 입금자명, 유니폼 사이즈, 수업 진행 여부(라디오/체크).
	•	성공 시 “신청 접수 완료” 페이지/모달 노출.

4) 제품 상세/결제 플로우를 게스트 결제로 통일
	•	제품 상세 페이지
	•	“로그인 필요” 문구/“로그인하러 가기” 버튼 모두 삭제
	•	주요 CTA를 “결제하기” 하나로 통일(Primary).
	•	클릭 시 바로 결제 모달/체크아웃 페이지로 이동.
	•	결제 페이지(Checkout)
	•	로그인 없이 결제 가능해야 한다.
	•	게스트 결제를 위해 아래 정보를 수집:
	•	수취인 이름, 연락처, 이메일(영수증 발송용), 배송지 주소
	•	(전자결제 필수 동의) 결제 약관/개인정보 제공 동의 체크박스
	•	결제 위젯(토스 등)에서 테스트/운영 설정은 기존 그대로 사용.
	•	결제 성공 시: 주문번호, 금액, 결제수단, 수취인/배송지 요약, 고객 안내 문구 표시
	•	결제 실패 시: 에러 코드/메시지 + 재시도 버튼

5) DB/주문 기록
	•	게스트 결제 시에도 주문 레코드를 저장한다.
	•	customerType: 'guest' 로 마킹
	•	고객 입력 정보(이름/연락처/이메일/주소), 라인아이템, 결제ID/상태/시각 저장
	•	관리자 메뉴 /admin/orders 에서 게스트 포함 최근 주문을 확인 가능하게 리스트 유지.

6) 관리자 전용 로그인/대시보드
	•	/admin/login 페이지는 그대로 유지(또는 생성).
	•	관리자만 진입할 수 있도록 보호.
	•	로그인 성공 시 /admin 대시보드로 이동(주문/세미나 신청 등 데이터 확인).

7) 화면 텍스트/UI 정리
	•	제품 상세/결제 전역에서 “로그인 필요” 관련 모든 문구/버튼 제거.
	•	결제 CTA는 반드시 “결제하기” 로 표시.
	•	헤더 네비: “관리자” 외 로그인/회원가입 링크가 보이지 않도록 숨김.
	•	푸터/정책: 게스트 구매/개인정보 활용/환불 정책 링크가 있다면 계속 유지.

8) 테스트 시나리오 (반드시 통과)
	1.	상단 네비에 **“관리자”**만 보이고, 클릭 시 /admin/login 으로 이동한다.
	2.	세미나 신청 페이지: 로그인 없이 폼 입력 → 제출 성공 → DB 저장/알림 정상.
	3.	제품 상세 페이지: “결제하기” 버튼만 보이고, 누르면 곧바로 결제창(모달/체크아웃) 열린다.
	4.	결제 페이지: 비로그인 상태에서 수취인 정보/주소 입력 → 결제 완료 → 주문번호 확인.
	5.	관리자 로그인 후 /admin/orders 에서 방금 게스트 주문이 보인다.
	6.	구매/신청 어느 경로에서도 로그인 유도 문구가 한 단어도 보이지 않는다.
